# kuja-build-resources

kuja 项目里编译打包使用的相关资源的项目。

目前包括了以下内容：

| 编译打包工具 | 设置所在目录    | 补充说明                                           |
| ------------ | --------------- | -------------------------------------------------- |
| check-style  | kuja/checkstyle |                                                    |
| spotbugs     | kuja/spotbugs   |                                                    |
| pmd          | kuja/pmd        | 主要的规则定义，都在项目 `kuja-p3c-pmd` 项目中定义 |
